# Abrigo Animal

Site institucional para abrigo de animais abandonados.

### Funcionalidades
- Página inicial com apresentação e carrossel de animais
- Página de adoção com filtros
- Página de apadrinhamento
- Página de doações
- Página de voluntários com formulário
- Página de contato
- Página sobre nós

### Pré-requisitos
- Node.js instalado na máquina

### Instalação
```bash
npm install
```

### Como Iniciar
```bash
npm start
```

A aplicação estará disponível em http://localhost:3000.
